# ResumeBuilderApp
This App has feature to create your own resume based on your input data.
User information fields are:
  1. Profile image
  2. Personal information
  3. Objective
  4. Total experience
  5. Education
  6. Skills
  7. Work summary
  8. Project details
  
  User can create, view and delete their resume.
